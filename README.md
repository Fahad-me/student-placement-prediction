# Student Placement Prediction

This project predicts whether a student will be placed or not based on:

- CGPA
- IQ Score

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-Learn
- Matplotlib

## Machine Learning Model
- Logistic Regression

## Dataset Features
- CGPA
- IQ
- Placement (0 = Not Placed, 1 = Placed)
